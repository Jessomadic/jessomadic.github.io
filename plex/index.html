One-time setup needed

Connect Firebase (free)

PickFlick needs Firebase Realtime Database to sync swipes between devices. Takes about 5 minutes.

1
Create a free Firebase project Visit console.firebase.google.com → Add project → name it anything → Create
2
Register a Web app Click the </> icon on the project overview → nickname: "pickflick" → Register app → copy the config values
3
Enable Realtime Database Build → Realtime Database → Create Database → Start in Test Mode → Enable
4
Set database rules In Realtime Database → Rules tab → paste and Publish:
{"rules":{"sessions":{"$c":{".read":true,".write":true}}}}
5
Paste your config into plex/firebase-config.js Replace the YOUR_* placeholders with your project's actual values, then push to GitHub.
